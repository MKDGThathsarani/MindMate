# 🧠 MindMate

### AI-Powered Mental Wellness Companion for Sri Lankan University Students

---

## 📱 About the Project

**MindMate** is a mobile application designed to provide **anonymous, stigma-free mental health support** for university students in Sri Lanka. With the rising mental health crisis among students due to academic pressure, economic instability, and social stigma, MindMate offers a safe space for students to seek help.

---

## ✨ Key Features

| Feature | Description |
|---|---|
| **Anonymous Mood Tracking** | Daily check-ins with emoji-based mood logging and notes |
| **AI Chatbot (Sinhala/English)** | 24/7 conversational support using Dialogflow |
| **Self-Help Resources** | Offline articles, videos, and guided meditations |
| **Peer Support Groups** | Anonymous chat rooms moderated by professionals |
| **Counselor Booking** | Connect directly with university counselors |
| **Emergency Helpline** | One-tap access to CCC Foundation, Sumithrayo |

---

## 🛠️ Technology Stack

| Component | Technology |
|---|---|
| **Mobile Framework** | React Native (Android + iOS) |

| **AI Chatbot** | Dialogflow ES (Sinhala + English) |
